🧪 Quick Test Workflow

Test the Complete System (5 minutes)

  1. Login as Kitchen Team → Go to Inventory Management
  2. Add Item: Tomatoes, Vegetable, 100 kg
  3. Login as Cook → Go to Purchase Orders
  4. Create PO: Tomatoes, 200 kg @ ₱10/kg = ₱2,000
  5. Approve & Deliver → Check inventory (should be 300 kg)
  6. Go to Weekly Menu Dishes
  7. Create Dish: Tomato Soup, using 10 kg Tomatoes
  8. Check inventory → Should be 290 kg (automatic deduction!)
  9. View History → See all transactions logged
